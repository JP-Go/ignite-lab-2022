# Event Platform

Plataforma de eventos criada no evento Ignite Lab 2022 da Rocketseat

# Tecnologias utilizadas

- React.js
- React Router
- Tailwindcss
- Autoprefixer
- Postcss
- GraphQl
- GraphCMS
- GraphQl-codegen
- Phosphor React
- Date fns
- Classnames
